<h1 align="center">LuaBotForDevelopers</h1>
O programa da Lua Bot for Developers foi feito para estudantes aprenderem e poderem participar(Se a vaga for aprovada) do projeto da Lua Bot for Developers

<h1 align="center">Oque é a Lua Bot?</h1>
A Lua Bot é uma Bot do Discord que tem a Personalidade de uma Shihtzu que adora morder e brincar, então ela veio pro Discord!
(https://github.com/TheLuaBot/LuaBot)

<h1 align="center">Sobre o programa LuaBotForDevelopers</h1>
O Programa da LuaBotForDevelopers, é um programa que as pessoas podem contribuir e ajudar a LuaBot! O Programa tem parceria com a @PerfectTeaBR , A Empresa Dona do AstroOuuBot, KatSim e etc...
Se quiser procurar sobre a @PerfectTeaBR acesse: https://perfecttea.squareweb.app/
